Proyecto de E-commerce

TechShopper

Este proyecto es una plataforma de comercio virtual para la venta de dispositivos electronicos, fue bautizado como "TechShopper" debido a su enfoque en la compra de tecnología. Cuenta con una arquitectura por capas que integra un frontend en C#, la interfaz de usuario fue diseñada con HTML, CSS y Bootstrap, un backend en Java que maneja la lógica de negocio y una base de datos en MySQL desplegada en AWS. Fue desplegada en la nube mediante AWS EC2.
